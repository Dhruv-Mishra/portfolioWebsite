# Favicon and Icon Generation Instructions

## Required Files:
To make this website production-ready, you need to add the following image files to the `public` folder:

### 1. **favicon.ico** (16x16, 32x32, 48x48)
- Standard browser favicon
- Place at: `public/favicon.ico`

### 2. **apple-touch-icon.png** (180x180)
- iOS home screen icon
- Place at: `public/apple-touch-icon.png`

### 3. **icon-192.png** (192x192)
- PWA icon for Android
- Place at: `public/icon-192.png`

### 4. **icon-512.png** (512x512)
- PWA icon for high-res displays
- Place at: `public/icon-512.png`

### 5. **og-image.png** (1200x630)
- Open Graph image for social media sharing
- Place at: `public/resources/og-image.png`

## Quick Generation:
You can use tools like:
- https://realfavicongenerator.net/
- https://favicon.io/
- Or design your own with your initials "DM" in the sketchbook style

## Recommended Design:
- Use the sketchbook/handwritten aesthetic from your site
- Consider using your initials "DM" in the Patrick Hand font
- Colors: Use your site's indigo accent (#4f46e5) on paper background (#fdfbf7)
